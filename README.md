<h1 align="center">PRODUCT MANAGEMENT</h1>

## Description

- This is a sample project for crud product and category.
- Main features:
  - Sign in, sign up
  - Crud Product, category
- Technologies: NestJS, GrapQL, TypeORM, MySQL

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
